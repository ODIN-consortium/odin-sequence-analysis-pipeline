# ODIN Sequence Analysis Pipelines

This repository uses separate branches for independent analysis pipelines.

The `main` branch is only an index. It does not contain an active pipeline and
should not be used for analysis runs.

## Pipeline branches

- `longread_analysis`: long-read sequence analysis pipeline.
- `wgs_analysis`: whole-genome sequencing analysis pipeline based on Bactopia.

Switch to the branch for the pipeline you want to use before running or editing
pipeline code.
